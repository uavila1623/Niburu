Start Search...

Please use the steps below to use the search function:

  1. Enter the weather type, city, county, or state that you are looking for in search bar.
  2. Then enter how many rows you want to display by default it displays 10 rows. Please note that the larger number of rows then the longer it takes to load results.
  3. After you entered all the fields press the search button and wait for the results to display.
  4. If any results are found then it will display them under the column names.
  5. You can use the row values as inputs to make a prediction or to create a chart.

Please note that due to the size of the data the search length is limited to 100 rows max.
