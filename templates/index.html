Nibiru

Welcome to Nibiru

Nibiru is a machine learning web app that I built to make different weather predictions based on data about past weather events. The data it is using is from a csv file that I downloaded from Kaggle which is an online community of data scientists and machine learning enthusiasts who come together to participate in various data science competitions, collaborate on data science projects, and share knowledge and insights about data science and machine learning. The purpose of building this app was to bring awareness about climate change by displaying how the weather has changed over the years. And to also allow users to experiment with some machine learning models and to see visualizations of those models. Currently the app can only make minor predictions because the data has limited numerical data to base the machine learning models on. Hopefully with further development or the use of different data I could integrate more prediction models.

To create a regional prediction please follow the following steps:

  1. First choose the state of the region you would like to use.
  2. Next choose the weather type you would like to base your prediction on.
  3. Then enter the city that your region is based in.
  4. After that enter the county that the city is in.
  5. Lastly enter 6 predictions, to help train the data.

Please note that your predictions influence the final prediction drastically. If you need help choosing the predictions please visit the charts page to gain more insight on the data being used.
